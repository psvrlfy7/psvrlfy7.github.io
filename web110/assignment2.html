Assignment 2: Web Browsers - Learning Summary

Activity 1: Install multiple web browsers to compare the different programs – Chrome

Topic: Comparing web browsers

What I learned:
While installing Google Chrome, I got a better idea of how different browsers work and what features they offer. Chrome is fast, easy to set up, and works smoothly with other Google services like Gmail and Google Drive. Compared to browsers like Safari or Edge, Chrome feels more flexible, especially with the wide range of available extensions and tools for developers.

How I can use this:
I can use Chrome as my main browser, especially when I need access to developer tools or specific browser extensions. Knowing how different browsers work is also useful for checking how websites look and behave across platforms, which is important for school or future web projects.

Activity 2: Practice using web browsers – macOS

Topic: Learning browser features on macOS

What I learned:
Using a browser on macOS helped me see how the interface and shortcuts are a little different from Windows. For example, Safari on Mac has its own way of managing tabs, downloads, and settings. I also practiced using features like private browsing, bookmarks, and browsing history. These small things help me browse more efficiently.

How I can use this:
In the future, if I work on a Mac or help someone who uses one, I’ll be more confident with the browser setup. It also makes me more flexible when switching between operating systems and helps me adjust quickly to different tech environments.

Activity 3: Configure browser settings – Chrome

Topic: Changing Chrome settings for better use

What I learned:
In this activity, I explored the settings menu in Chrome. I changed the default search engine, looked at how to manage saved passwords, and updated privacy settings. I also learned how to clear cache and cookies and block pop-ups. These features help make browsing safer and more comfortable.

How I can use this:
I can apply this by customizing Chrome for better performance and privacy. For example, I can turn off tracking, block annoying pop-ups, and make sure my passwords are safe. This is especially helpful when I work on shared or public computers.
