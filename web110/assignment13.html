WEB110 Assignment 13 - Databases

Example of single-table SELECT statements:

  • SELECT * FROM CUSTOMERS;
  • SELECT CustomerName, ContactName FROM CUSTOMERS;
  • SELECT CustomerName, City FROM CUSTOMERS WHERE Country='Germany';

Example of INSERT, UPDATE, and DELETE statements:

  • INSERT INTO CUSTOMERS (CustomerName, ContactName, Country) VALUES ('New Customer', 'John Doe', 'USA');
  • UPDATE CUSTOMERS SET ContactName='Jane Doe' WHERE CustomerID=1;
  • DELETE FROM CUSTOMERS WHERE CustomerID=10;

During this assignment, I learned how to write basic SQL statements for selecting, inserting, updating, and deleting data in a database. Understanding these operations allows me to manage and manipulate data efficiently, which is essential for real-world web development and database management. I can apply this knowledge in any project that requires storing and retrieving structured data.
