Types of CSS and How They Work

CSS, or Cascading Style Sheets, is used to style web pages. There are three main types of CSS: external, internal, and inline.

1. External CSS

External CSS is stored in a separate .css file and linked to an HTML page using the tag. This method is useful when you want to apply the same style to multiple pages. Example: In a file called style.css you could write:
  • h1 { color: red; }
  • p { font-size: 16px; }
Then link it to all your HTML pages to apply the same styling everywhere.

2. Internal CSS

Internal CSS is written inside the <style> tag within the <head> section of an HTML page. It applies only to that specific page and has higher priority than external CSS but lower than inline CSS.

Example:

    <head>
      <style>
        h1 { color: blue; text-align: center; }
        p { font-size: 18px; }
      </style>
    </head>

This will make the heading blue and centered, and the paragraph slightly larger, but only on this page.

CSS Priority and Cascading

CSS follows a priority order, also called “cascading”:

  • Inline CSS (highest priority)
  • Internal CSS (medium priority)
  • External CSS (lowest priority)

This means that if multiple styles target the same element, the style with higher priority will be applied.

3. Inline CSS

Inline CSS is applied directly to an HTML element using the style attribute. It has the highest priority and overrides other styles.

Example:

style="color: green; font-size: 22px;"This text will appear green."

CSS Selectors and Properties

CSS uses selectors to target elements and properties to define how they look.

  • Selector example: h1, p, .classname, #id
  • Property example: color, font-size, text-align, background-color, margin

Example: h1 { color: red; text-align: center; font-size: 24px; }

This makes the heading red, centered, and larger.

Conclusion and how to Apply

By using external, internal, and inline CSS, and choosing the correct selectors and properties, you can create visually appealing, consistent, and responsive web pages. You can apply this learning to any website project to control layout, fonts, colors, and overall design.
